TinyOS 2.x Documentation
Last Modified: Oct 31 2005

There are three classes of TinyOS 2.x documentation. The first class includes release notes and high-level overviews of the system. The second is a series of tutorials. The first is a set of TEPs (TinyOS Enhancement Proposals), which document and describe major OS abstractions and subsystems.

As TinyOS 2.x is still in prerelease, there are not yet any tutorials.

TEPs
TEPs are written in ReStructured Text, and so can be read either as HTML or as a text document. TEP 1 describes the different kinds of TEPs and their roles. TEPs 1-100 are BCP (Best Common Practice) TEPs, while TEPS 101+ are Informational, Documentary, or Experimental. Currently, all TEPs are still Drafts: comments and feedback to the authors (or the 2.x working group) is welcome.
